AWS CLI 命令行工具
=================

## AWS CLI是什么

AWS Command Line Interface (AWS CLI) 是一种开源工具，
让您能够在命令行 Shell 中使用命令与 AWS 服务进行交互。
仅需最少的配置，即可使用 AWS CLI 开始运行命令，以便从终端
程序中的命令提示符实现与基于浏览器的 AWS 管理控制台所提供的
功能等同的功能。
