## RDS定义

RDS是被管理的数据库服务，让开发人员免去系统(OS,DB)管理的麻烦，重点把精力放在业务逻辑的开发。

## RDS支持的数据库引擎

+ Amazon Aurora(MySQL, PostgreSQL)
+ MySQL
+ MariaDB
+ PostgreSQL
+ Oracle
+ SQL Server

## 优点

+ 轻松管理
+ 高度可扩展
+ 可用且持久
+ 速度快
+ 安全
+ 经济实惠(考虑人员维护成本)
